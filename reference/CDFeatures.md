# Chemical Descriptor Features

Vector containing the feature names of the chemical descriptors listed
in [CDNames](https://spang-lab.github.io/FastRet/reference/CDNames.md).

## Usage

``` r
CDFeatures
```

## Format

An object of class `character` of length 241.

## See also

[CDNames](https://spang-lab.github.io/FastRet/reference/CDNames.md)

## Examples

``` r
str(CDFeatures)
#>  chr [1:241] "Fsp3" "nSmallRings" "nAromRings" "nRingBlocks" "nAromBlocks" ...
```
