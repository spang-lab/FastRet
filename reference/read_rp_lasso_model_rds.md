# LASSO Model trained on RP dataset

Read a LASSO model trained on the
[RP](https://spang-lab.github.io/FastRet/reference/RP.md) dataset using
[`train_frm()`](https://spang-lab.github.io/FastRet/reference/train_frm.md).

## Usage

``` r
read_rp_lasso_model_rds()
```

## Value

A `frm` object.

## Examples

``` r
frm <- read_rp_lasso_model_rds()
```
